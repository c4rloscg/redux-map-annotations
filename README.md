# Map annotations app

This project is an example of how to use annotations in a WebGL based map such as Maplibre or Mapbox.
Libraries:

- React.js
- Maplibre-gl
- Tailwind CSS
- Redux toolkit (RTK)

## Getting Started

To run this app locally:

1. Run `yarn`
1. Run `yarn start`

The app will start on http://localhost:3000
